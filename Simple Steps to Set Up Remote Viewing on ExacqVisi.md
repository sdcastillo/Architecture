## Simple Steps to Set Up Remote Viewing on ExacqVision NVR

1. **Install the ExacqVision Client**
    - Download and install the ExacqVision Client software on the remote computer from the Exacq website[^2][^4].
2. **Get the NVR’s IP Address**
    - Find the NVR’s IP address on your network. You may need your network admin to help with this[^2][^3].
3. **Connect Using the Client**
    - Open the ExacqVision Client.
    - Click to add a new system.
    - Enter the NVR’s IP address, username, and password[^2][^4].
4. **(For Offsite Viewing) Set Up Port Forwarding**
    - If you want to view the NVR from outside your building, set up port forwarding on your router for the NVR’s ports (ask your IT or network admin if unsure)[^3].
5. **Test the Connection**
    - In the client, check that you can see live video from the NVR. If not, double-check the IP address, credentials, and network connection[^2][^4].

That’s it-remote viewing is ready. For even easier setup, use the Exacq cloud-hosted relay service if available, which skips most network steps[^1].
